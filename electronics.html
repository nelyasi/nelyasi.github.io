00 — Background

Where it all started

C/C++ → microcontroller → motion  ·  the original loop that hooked me on computers

Long before quantum mechanics, there were microcontrollers. Programming embedded systems in C and C++ as a teenager is what first made computing feel tangible and real. These hobby projects are where that curiosity still lives — physical electronics that you can hold, break, and fix.

Line-Following Robot

The project that started everything. A two-wheeled robot that reads an IR sensor array and uses a PID controller to follow a black line on white paper. Built with an AVR microcontroller, written in pure C.

ESP32 Weather Station

A low-power ESP32-based station measuring temperature, humidity, and atmospheric pressure. Data is logged over Wi-Fi to a local server and visualised with a small OLED display. Runs on a LiPo cell with solar top-up.
